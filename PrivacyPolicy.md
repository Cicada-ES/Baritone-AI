# Baritone AI Privacy Policy

Effective Date: November 8, 2025

Baritone AI respects your privacy. By using this bot, you consent to the collection and use of information as described below.

## Information Collected
- **User IDs and tags:** For moderation purposes (warnings, mutes, kicks, bans)
- **Timestamps:** To track when actions occur
- **Server IDs and names:** To link moderation data to specific servers

No other personal information is collected. Baritone AI does **not** sell, share, or distribute user data.

## Data Storage
- Moderation data is stored securely in Firebase Firestore.
- Only authorized moderators can view user histories using bot commands.

## Security
- Data is protected by Firebase security protocols.
- Unauthorized access is prohibited.

## Data Retention & Deletion
- Data is retained for moderation purposes.
- Server owners may request data removal by contacting Kn3ghtfall.

## Changes
- Privacy policy may be updated; users are encouraged to review periodically.
