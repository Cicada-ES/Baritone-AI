# Baritone AI

Baritone AI is a Discord moderation bot designed for robust, persistent server management.

## Features
- Warn, mute, kick, and ban users
- View moderation history
- Server and user info commands
- Admin-only bot lock/unlock
- Multi-unit mute durations (e.g., `1h30m`, `2d`)
- Persistent data storage via Firebase
- Full moderation logging for accountability

## Commands
| Command | Description |
|---------|-------------|
| `?warn` | Warn a user |
| `?unwarn` | Remove last warning |
| `?mute` | Mute a user with duration |
| `?unmute` | Remove mute |
| `?kick` | Kick a user |
| `?ban` | Ban a user |
| `?unban` | Unban a user |
| `?view` | View user history |
| `?serverinfo` | Show server info |
| `?userinfo` | Show user info |
| `?lock` | Lock bot for admins only |
| `?unlock` | Unlock bot |
| `?help` | Display this message |
